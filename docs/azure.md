# Azure

## Configuration

### Example

```json
{
  "settings": [
    {
      "provider": "azure",
      "domain": "domain.com",
      "host": "@",
      "subscription_id": "",
      "resource_group_name": "",
      "zone_name": "",
      "relative_record_set_name": ""
    }
  ]
}
```

### Compulsory parameters

- `"domain"`
- `"host"` is your host and can be a subdomain or `"@"`
- `"subscription_id"`
- `"resource_group_name"`
- `"zone_name"`
- `"relative_record_set_name"`
